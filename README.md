# Credit-Risk-Analysis-Page
## Features

- Financial ratio analysis
  - Current Ratio
  - Working Capital
  - Profit Margin
  - EBITDA Margin
  - Return on Assets (ROA)
  - Debt-to-Assets Ratio
  - Debt-to-EBITDA Ratio
  - Debt Service Coverage Ratio (DSCR)

- Risk scoring engine
  - Weighted 0–100 credit risk model
  - Industry-specific risk adjustments
  - Transparent score breakdown by category

- Lending recommendations
  - Approve
  - Approve with Conditions
  - Further Review Required
  - Decline

- Executive dashboard
  - KPI cards
  - Risk gauge
  - Financial health overview
  - Underwriting metrics and visualizations

- Credit analyst insights
  - Key strengths
  - Key weaknesses
  - Underwriting concerns
  - Risk mitigants
  - Recommendation rationale

## Technologies

- HTML
- CSS
- JavaScript
